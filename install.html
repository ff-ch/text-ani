text-ani

Animate text and export it as a video with a transparent background — ready to drop straight into Final Cut, Premiere or After Effects.

Let’s get text-ani onto your Mac.

Three steps, about two minutes.

Set it up — 3 steps

1

Open the Terminal app

“Terminal” is a small app that comes built into every Mac. To open it:

Hold down the ⌘ Command key and tap the Space bar. A search box appears — type the word Terminal and press Return.

A plain window with a blinking cursor opens. That’s Terminal — leave it open for the next step.

2

Copy the command

Click the white Copy button.

curl -fsSL https://raw.githubusercontent.com/ff-ch/text-ani/main/install.sh | bash

The command downloads the text-ani script and the tools it uses to make video (ffmpeg and Node). See what it does.

3

Paste it in and press Return

Click the Terminal window, press ⌘ Command + V, then Return. Then follow along:

  • A window asks where to save text-ani — choose any folder you like.
  • If it asks for your Mac login password, type it and press Return (you won’t see it as you type).
  • Wait a minute — text-ani then opens in your browser, ready to use.

That’s it 🎉

To open text-ani again another day, go to the folder you chose, open the text-ani folder, and double-click text-ani.command.

I’m on Windows or Linux

Linux: install Node.js and ffmpeg using your system’s software installer, then run:

git clone https://github.com/ff-ch/text-ani && cd text-ani && npm install && npm start

Windows: install Node.js and ffmpeg, then open the project folder, run npm install followed by npm start, and visit http://localhost:4444 in your browser.

I’m comfortable with the command line and want to do it myself

1. Install Homebrew, then brew install node ffmpeg

2. git clone https://github.com/ff-ch/text-ani

3. cd text-ani && npm install && npm start → open http://localhost:4444
