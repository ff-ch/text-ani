text-ani

Animate text in your browser → export ProRes 4444 with a real alpha channel.

Install on macOS detecting…

Paste this one line into Terminal and press Return. It sets everything up and opens the app.

curl -fsSL https://raw.githubusercontent.com/ff-ch/text-ani/main/install.sh | bash

It will:

  1. install Homebrew if you don't have it (it may ask for your Mac password),
  2. install ffmpeg and Node.js,
  3. download text-ani to ~/text-ani, and
  4. launch it in your browser.
No “unidentified developer” prompt — you're running a script you can read, not opening a downloaded app. The full script is on GitHub if you'd like to check it first.

To open Terminal: press ⌘ Space, type “Terminal”, press Return.

On Windows or Linux?

Linux — install Node.js and ffmpeg with your package manager, then:

git clone https://github.com/ff-ch/text-ani && cd text-ani && npm install && npm start

Windows — install Node.js and ffmpeg (add ffmpeg to your PATH), then in the project folder run npm install and npm start, and open http://localhost:4444.

Prefer to do it by hand?
  1. Install Homebrew, then run brew install node ffmpeg
  2. git clone https://github.com/ff-ch/text-ani
  3. cd text-ani && npm install && npm start
  4. Open http://localhost:4444
